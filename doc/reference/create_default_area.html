create_default_area

create_default_area(
floor:Floor,
name:str|None = None,
sample_num:int = 300
)->Area

概要

Areaインスタンスを作成します。
捜索範囲以外（初期動作確率、遷移確率、利用者位置分布、遷移失敗確率）はデフォルト値になります。

引数

名称 型 概要
floor Floor 捜索範囲
name str|None 名前（メタデータ）
sample_num int 遷移失敗確率のサンプル数

戻り値

型：Area
概要：
作成されたAreaインスタンス

使用例

from wilitools import Floor, create_default_area

floor = Floor(-5.0, 5.0, -5.0, 5.0)
area = create_default_area(floor, sample_num=10)

print(area)
