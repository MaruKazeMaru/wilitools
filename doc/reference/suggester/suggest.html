Suggester.suggest

Suggester.suggest(
x:numpy.ndarray
)->numpy.ndarray

概要

なくしもの位置の推定を行います。
推定結果は次のような式で表されます。

suggest関数は引数としてxを渡し、戻り値としてfを得ます。

引数

名称 型 概要
x numpy.ndarray 推定結果の変数（上式のx）
配列の次元は任意です。ただし最後の次元の要素数は2である必要があります。
例えば配列の次元が(5,3,2)や(2)の場合は問題ありませんが、(5,3,3)の場合はエラーが発生します。

戻り値

型：numpy.ndarray
概要：
推定結果の値（上式のf）です。
引数xが1次元配列ならばfloat32に、m次元配列ならばm-1次元配列のndarrayになります。

使用例1

引数xが1次元配列（＝ベクトル1個）の場合の例です。 
from wilitools import create_default_area, area_to_suggester, Floor
import numpy as np

floor = Floor(-5, 5, -5, 5)
area = create_default_area(floor)
suggester = area_to_suggester(area)

x = np.array([0, 0])
print('x=')
print(x)

f = suggester.suggest(x)
print('f=')
print(f)

実行結果

x=
[0 0]
f=
0.00030724137

使用例2

引数xが2次元配列（＝ベクトル複数個）の場合の例です。 
from wilitools import create_default_area, area_to_suggester, Floor
import numpy as np

floor = Floor(-5, 5, -5, 5)
area = create_default_area(floor)
suggester = area_to_suggester(area)

x = np.linspace([0,-2], [0,2], 5) # changed this line
print('x=')
print(x)

f = suggester.suggest(x)
print('f=')
print(f)

実行結果

x=
[[ 0. -2.]
 [ 0. -1.]
 [ 0.  0.]
 [ 0.  1.]
 [ 0.  2.]]
f=
[0.00301858 0.00111824 0.00030724 0.00116728 0.00315281]
