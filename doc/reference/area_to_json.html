area_to_json

area_to_json(
json_path:str,
area:Area,
miss_probs_in:bool = False,
json_dump_kwargs:dict = {}
)->None

概要

AreaインスタンスをJSONファイルに出力します。

引数

名称 型 概要
json_path str 出力先JSONファイルのパス
area Area 出力元Areaインスタンス
miss_probs_in bool 遷移失敗確率のサンプルを出力するか
json_dump_kwargs dict JSONファイルに出力する際のオプション
内部でjson.dump関数を呼び出しており、json_dump_kwargsはその関数に渡すオプション引数です。json.dumpのオプション引数についてはPython公式ドキュメント等をご参照ください。
Python公式ドキュメント：https://docs.python.org/ja/3/library/json.html

戻り値

なし

使用例

from wilitools import json_to_area, area_to_json
