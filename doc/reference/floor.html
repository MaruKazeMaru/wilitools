Floor

概要

捜索範囲を表すクラスです。
wilitoolsでは捜索範囲をR2中の長方形と仮定しています。そこで

メンバ変数

名称 型 概要
x_min float x方向の最小値
x_miax float x方向の最大値
y_min float y方向の最小値
y_max float y方向の最大値

コンストラクタ

Floor(
x_min:float,
x_max:float,
y_min:float,
y_max:float
)

引数

名称 型 概要
x_min float x方向の最小値
x_max float x方向の最大値
y_min float y方向の最小値
y_max float y方向の最大値

備考
x_minがx_maxより大きい、またはy_minがy_maxより大きい場合はValueErrorを発生させます。

メンバ関数

名称 概要
get_lattice 捜索範囲内の格子点を取得
格子の間隔は引数で指定
